# 射影几何

计算机中最常用的是 2D 和 3D 射影几何. 

## 2D 射影几何

注意: 不同几何中的各种概念存在差异. 射影几何中的无穷远点和无穷远线, 在欧氏几何中并不是点或者线.

点和线的概念对偶. 交换定理中的点与线的概念, 依然成立.

引入了无穷远点和无穷远线, 所有无穷远点都在无穷远线上.

- 有限点: $[x,y,1]$
- 无穷远点: $[x,y,0]$
- 普通直线: $[a,b,c]\ (ab \ne 0)$
- 无穷直线: $[0,0,c]$

引入无穷远点和无穷远线之后, 两条相异直线交于一点, 两相异点恰在一条直线上. 因此, 可以根据两条线确定一个点, 也可以根据两个点确定一条直线.

点在直线上的充要条件: 内积为0(商结构下成立).

射影几何中的二次曲线, 由3x3对称矩阵确定, 因此二次曲线有五个自由度. 一条二次曲线可以由(一般位置的)五个点唯一确定.

点在二次曲线上: xTCx = 0

(非退化)二次曲线的切线由切点确定: l=Cx

对偶二次曲线由一族线组成, 也被称为二次曲线的包络, 和点二次曲线一样, (一般位置上的)五条线确定一条对偶二次曲线.

退化二次曲线: 由降秩矩阵C定义的二次曲线

Erlangen Program: 几何研究在变换群下保持不变的性质.

2D射影几何研究射影平面在射影映射的变换群下保持不变的性质.

射影映射的几何定义: 将共线的点,映到同一条线的**可逆**映射.

射影映射也叫保线变换, 射影变换或者单应(homography).

射影映射的代数定义: 映射可以表示成Hx, 其中H是非奇异矩阵.

代数定义完全描述了射影映射的形式.

显然,射影映射有8个自由度.

映射显然还可以作用于直线和二次曲线.

点变换是逆变的, 而线的变换是协变的.

射影映射群称为射影线性群, 记为PL(n), 它是GL(n)的商群

PL(3)的重要子群包括仿射群和欧式群. 仿射群的元素最后一行是(0,0,1), 欧式群是仿射群的子群, 左上的2x2矩阵是正交的. 当2x2矩阵行列式为1时,称为定向欧式群.

某些有趣的变换不构成群.

有的元素或者量在变换下保持不变, 称为不变量. 研究不变量也是研究变换群的一种方法.

距离在欧式变换(平移和旋转)下不变, 在相似变换下不是不变量.

夹角是欧式和相似不变量.

等距变换保持欧式距离不变. 等距变换分为保向和逆向的.
